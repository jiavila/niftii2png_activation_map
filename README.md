# niftii2png_activation_map

## Description
Creates single x, y, and z viewpoint plots of nifti functional MR images of
the brain and saves them as a single PNG (.png) file.
 
This gear uses the [NiBabel](https://nipy.org/packages/nibabel/index.html) 
package to open nifti files and then uses the Neuroimaging in Python ([NiPy](https://nipy.org/packages/nipy/index.html)
) package to create the activation map plot. This currently supports only 3D
 volumes of the brain and not 4D (timeseries) nifti files. 
 


## Inputs

#### nifti: file (required)

A 3D functional magnetic (fMR) resonance image of the brain in nifti format
 ('.nii.gz' or 'nii').

#### fig_rez: string (optional)

The size of the final output .png image, this sets the figure size. This can
either be '1080p' for creating a 1920x1080 image or '720p' for creating a
1280x720 image. The default value is '720p' if none is passed.

## Outputs
#### png_image: file

The .png image of the x, y, and z viewpoint plots of brain fMR image, it is
saved with the same name as the input nifti file but with the '.png' extension. 
This is store in the default Flywheel output directory (`flywheel/v0/output`)
